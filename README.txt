PayItMonthly Payment gateway Wordpress plugin for Woocommerce

== Description ==

This plugin made to add an additional payment option to WooCommerce plugin.
You can find more details here: <a href="https://payitmonthly.uk/integration/payitmonthly-for-woocommerce-plugin/">https://payitmonthly.uk/integration/payitmonthly-for-woocommerce-plugin</a>


= External Service Provider =
We are an external service provider (https://payitmonthly.uk). We provide your customers the ability to spread their payment over a number of instalments. In order for us to provide the service to you and your customers this plugin will pass data to PayItMonthlys servers where it will be processed in accordance with our Privacy Policy.

= Privacy Policy =
Is available here - https://payitmonthly.uk/legals/privacy-policy/

== Installation ==

1. Install plugin through the Wordpress admin or upload pim_wc_payment_integration folder to the `/wp-content/plugins/` directory
2. Activate the plugin in Wordpress admin
3. Enable PayItMonthly payment gateway in WooCommerce payments settings
4. Add your Api Key and Api Password and configure PayItMonthly payment gateway settings
